= Reprise après incident de SAP HANA avec la réplication du stockage
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Ce document présente un aperçu des options de protection contre les sinistres pour SAP HANA. Il inclut des informations détaillées sur la configuration et une description d'un cas d'utilisation d'une solution de reprise après sinistre à trois sites, basée sur la réplication de stockage synchrone et asynchrone NetApp SnapMirror. La solution décrite utilise NetApp SnapCenter avec le plug-in SAP HANA pour gérer la cohérence de la base de données.

Auteur: Nils Bauer, NetApp
