= Découvrez comment configurer SnapCenter pour les environnements SAP HANA.
:allow-uri-read: 


[role="lead"]
Configurez SnapCenter pour les environnements SAP HANA en utilisant une approche en deux phases : configuration initiale pour les ressources partagées (informations d’identification, systèmes de stockage et stratégies) et configuration spécifique aux ressources pour chaque système HANA (déploiement de l’hôte, découverte automatique et paramètres de protection).

La configuration de SnapCenter pour un environnement SAP HANA avec plusieurs systèmes HANA peut être divisée en deux zones principales :

* La configuration initiale
+
** Configurations des identifiants, du stockage et des politiques. + Ces paramètres ou ressources sont généralement utilisés par plusieurs systèmes HANA.


* La configuration spécifique aux ressources HANA
+
** La configuration de l'hôte, de HANA et de la protection des ressources doit être effectuée individuellement pour chaque système HANA.

Toutes les étapes de configuration sont décrites en détail dans les rubriques suivantes.
