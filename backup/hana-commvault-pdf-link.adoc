= Sauvegarde et restauration SAP HANA à l'aide de systèmes de stockage NetApp et du logiciel Commvault
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Ce document décrit la conception d'une solution NetApp et Commvault pour SAP HANA, qui inclut la technologie de gestion des snapshots Commvault IntelliSnap et la technologie Snapshot NetApp. La solution est basée sur le stockage NetApp et la suite de protection des données Commvault.

Auteurs : Marco Schoen, NetApp; Dr Tristan Daude, CommVault Systems
